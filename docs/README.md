# Job Hunting OS

A production-ready Windows desktop application built with JavaFX to manage your career search.

## Features
- Complete Application, Company, and Interview tracking
- Interactive Kanban Board for Tasks
- Live Analytics Dashboard
- Secure local SQLite database
- Privacy-first: 100% offline
- Automated Zip Backups
- Data Export to CSV, Excel, and PDF

See the [User Guide](USER_GUIDE.md) for usage instructions.
