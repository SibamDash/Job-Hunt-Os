# User Guide

## Dashboard
The Dashboard displays your core metrics (Offers, Rejections, Interviews) which update automatically in real-time as you log applications.

## Adding Applications
Navigate to **Applications** -> **Add Application**. 
The system requires a Role and Company. Salarires can be entered as strings (e.g. "120k").

## Analytics
Use the filters at the top of the Analytics screen to narrow down statistics by Date Range or specific Roles.

## Backups
Go to **Settings** -> **Data & Backup**.
Click **Backup Now** to generate a complete Zip archive of your database. Do this weekly!
