# Installation Guide

## Requirements
- Windows 10/11
- 4GB RAM
- 50MB Disk Space

## Installation via Executable (Recommended)
1. Download JobHuntingOS.msi from the releases page.
2. Double-click the installer and follow the prompts.
3. Launch from your Start Menu.

## Portable Version
1. Download JobHuntingOS_Portable.zip.
2. Extract the folder to any location on your PC.
3. Run JobHuntingOS.exe directly.
